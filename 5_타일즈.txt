mvnrepository.com

1) tiles-extras 검색 > Tiles Extras(첫번째꺼) > 3.0.8

2) tiles-servlet 검색 > Tiles Servlet Support(첫번째거) > 3.0.8

3) tiles-jsp 검색 > Tiles JSP Support(첫번째꺼) > 3.0.8

pom.xml > dependencies태그 안에 추가


2. servlet-context.xml

1) 기존에 있던 <beans:bean class="....InternalResourceViewResolver">에
	<beans:property name="order" value="2" /> 추가

2) 아래에
<!-- tiles 설정 시작 
	TilesConfigurer tilesConfigurer = new TilesConfigurer();
	tiles-config.xml : 타일즈 설정 파일
	-->
	<beans:bean id="tilesConfigurer"
			class="org.springframework.web.servlet.view.tiles3.TilesConfigurer">
		<beans:property name="definitions">
			<beans:list>
				<beans:value>/WEB-INF/spring/tiles-config.xml</beans:value>
			</beans:list>
		</beans:property>
	</beans:bean>
	
	<beans:bean id="tilesViewResolver" 
			class="org.springframework.web.servlet.view.UrlBasedViewResolver">
		<beans:property name="viewClass"
				value="org.springframework.web.servlet.view.tiles3.TilesView" />
		<beans:property name="order" value="1" />
	</beans:bean>
	<!-- tiles 설정  끝  -->
추가

3) WEB-INF 아래 spring폴더에 xml파일 생성. 파일명: tiles-config.xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE tiles-definitions PUBLIC 
"-//Apache Software Foundation//DTD Tiles Configuration 3.0//EN" 
"http://tiles.apache.org/dtds/tiles-config_3_0.dtd">
<!-- tiles 설정 -->
<tiles-definitions>
   <!-- main layout -->
   <definition name="tiles-layout" template="/WEB-INF/views/tiles/index.jsp">
      <put-attribute name="header" value="/WEB-INF/views/tiles/header.jsp" />
      <put-attribute name="aside" value="/WEB-INF/views/tiles/aside.jsp" />
      <put-attribute name="body" value="" />
      <put-attribute name="footer" value="/WEB-INF/views/tiles/footer.jsp" />
   </definition>
   <!-- name의 */*은 body의 value에서 1과 2에 해당
      controller에서 forwarding 시 return "1/2";
    -->
   <definition name="*/*" extends="tiles-layout">
      <put-attribute name="body" value="/WEB-INF/views/{1}/{2}.jsp" />
   </definition>
</tiles-definitions>
붙여넣기


index.jsp의 틀을 기준으로
각 태그의 header, aside, footer의 경로에 있는 파일이 각 레이아웃으로 적용됨
